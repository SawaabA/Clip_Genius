Database Design

This database design organizes sports, matches, videos, users, comments, ratings, and tags into separate tables. Each table has a unique ID and links to others using foreign keys. For example, matches link to sports, videos link to matches, and comments/ratings link to videos and users. Tags are assigned to videos through a separate table. The design avoids duplicate data, ensures accuracy, and supports features like adding sports, uploading videos, tagging, commenting, and rating. It is simple, efficient, and scalable.

Database Schema

The database schema consists of the following tables:

  1. Sports: Stores information about sports.

  2. Users: Manages user accounts and roles. Synced With Original Providers Users Database

  3. Matches: Stores details about matches for specific sports.

  4. Videos: Stores video metadata and file paths.

  5. Tags: Manages tags for categorizing videos.

  6. VideoTags: Links videos to tags.

  7. Comments: Stores user comments on videos.

  8. Ratings: Stores user ratings for videos.

API Design

The API endpoints for managing a sports-related database. The API allows users to perform basic operations like creating, reading, updating, and deleting data for sports, users, matches, videos, tags, comments, and ratings.

API Endpoints

Resource HTTP Method Endpoint Description
Sports GET /sports Get all sports.
GET /sports/{sportId} Get a single sport by ID.
POST /sports Create a new sport.
PUT /sports/{sportId} Update a sport by ID.
DELETE /sports/{sportId} Delete a sport by ID.
Users GET /users Get all users.
GET /users/{userId} Get a single user by ID.
POST /users Create a new user.
PUT /users/{userId} Update a user by ID.
DELETE /users/{userId} Delete a user by ID.
Matches GET /matches Get all matches.
GET /matches/{matchId} Get a single match by ID.
POST /matches Create a new match.
PUT /matches/{matchId} Update a match by ID.
DELETE /matches/{matchId} Delete a match by ID.
Videos GET /videos Get all videos.
GET /videos/{videoId} Get a single video by ID.
POST /videos Create a new video.
PUT /videos/{videoId} Update a video by ID.
DELETE /videos/{videoId} Delete a video by ID.
Tags GET /tags Get all tags.
POST /tags Create a new tag.
DELETE /tags/{tagId} Delete a tag by ID.
Comments GET /videos/{videoId}/comments Get all comments for a video.
POST /videos/{videoId}/comments Create a new comment for a video.
DELETE /comments/{commentId} Delete a comment by ID.
Ratings GET /videos/{videoId}/ratings Get all ratings for a video.
